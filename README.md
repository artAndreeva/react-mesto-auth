# Место

Адаптивная верстка интерактивной страницы пользователя.

## Технологии

* Flexbox
* Grid
* Git
* Bash
* BEM
* Nested BEM
* DOM
* JS
* ООП
* Webpack
* API
* React

## Инструкция по запуску проекта

* Клонируйте проект
* Цстановите зависимости npm install
* Запустите npm start
* Локальные адрес http://localhost:3000
